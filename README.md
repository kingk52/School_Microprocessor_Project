# ⚙️ Microprocessor Project

## 📌 Overview
This project was developed as part of a Microprocessors course and focuses on low-level programming and hardware interaction.

The objective was to design and implement a system using a microprocessor, demonstrating an understanding of registers, memory, and instruction-level programming.

---

## 🧠 Functionality
- Execution of low-level instructions using assembly language
- Interaction with hardware components (e.g., input/output handling)
- Data processing using registers and memory
- Control flow implementation (loops, conditions, branching)

---

## 🚀 Features
- Assembly-level program design
- Direct manipulation of registers and memory
- Input/output handling through microprocessor instructions
- Efficient use of limited system resources
- Debugging at instruction level

---

## 🛠️ Technologies Used
- C++
- Microprocessor simulator / hardware kit ( Arduino)
- Low-level debugging tools
